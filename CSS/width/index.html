CSS width 属性

背景知识

  • 盒子模型分为四个区域 content、padding、border、margin
  • 标准模型下（box-sizing: content-box）元素的 width 属性设置的是 content 的宽度
  • IE模型下（box-sizing: border-box）元素 width 属性设置的是 content + padding + border 的宽度

width: 设置为auto，元素占据空间不超过容器，元素内容自动收缩

容器宽度为 800px，减去margin距离，实际盒子模型大小为780px

标准盒子模型下：width(auto) = content(auto)

box width(780px) = (content)(auto) + padding + border

content width: 780 - 10 * 2 - 2 * 2 = 756px

设置了 box-sizing: border-box

虽然现在是IE模型，但是因为width: auto，所以仍符合上面规则，盒子模型大小是780px

IE盒子模型下：width(auto) = (content + padding + border)(auto)

box width(780px) = (content + padding + border)(auto)

content width: 780 - 10 * 2 - 2 * 2 = 756px

padding, border 是固定值，所以不管哪种模型使用 width：auto，伸缩的都是content区域；
